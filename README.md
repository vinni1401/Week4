## üìù Overview

In Week 4, we focus on:

- **Java Generics**: Writing type-safe and reusable code.
- **Collections Framework**: Efficient data storage and manipulation.
- **Java I/O Streams**: Reading and writing data effectively.
- **Exception Handling**: Building robust applications with proper error management.
- **Regular Expressions and JUnit Testing**: Validating data and ensuring code reliability.
- **Java 8 Features**: Leveraging modern Java functionalities for cleaner code.

---

## üìò Topics Covered

### Day 1: Java Generics
- Understanding the need for generics.
- Implementing generic classes and methods.
- Bounded type parameters and wildcards.

### Day 2: Collections Framework
- Overview of List, Set, and Map interfaces.
- Implementations: ArrayList, LinkedList, HashSet, TreeSet, HashMap.
- Iterating over collections using iterators and for-each loops.

### Day 3: Java I/O Streams
- Byte streams vs. character streams.
- Reading from and writing to files.
- Buffered streams and their advantages.

### Day 4: Exception Handling
- Types of exceptions: checked and unchecked.
- Try-catch-finally blocks.
- Creating custom exceptions.

### Day 5: Regular Expressions and JUnit Testing
- Pattern matching with regular expressions.
- Validating input data.
- Writing unit tests using JUnit framework.

### Day 6: Java 8 Features
- Lambda expressions and functional interfaces.
- Stream API for processing collections.
- Method references and default methods.
